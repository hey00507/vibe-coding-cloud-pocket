## 구현 내용: StatisticsScreen 차트 시각화 추가

엑셀 템플릿(2026 시트)의 차트 2종을 앱에 구현.

### 인프라 설정
- `react-native-svg` 패키지 설치 (Expo 54 호환)
- `jest.setup.js`에 SVG 컴포넌트 모킹 추가 (Svg, Circle, Rect, Text, G, Line 등)

### DonutChart 컴포넌트
- SVG `<Circle>` strokeDasharray/strokeDashoffset 기법으로 도넛 차트 구현
- 카테고리별 지출 비율을 시각적 아크 세그먼트로 표현
- 중앙 텍스트 (총 지출 라벨 + 금액)
- 하단 범례 (색상 + 라벨 + 비율%)
- 빈 상태 처리 (회색 원 + "데이터가 없습니다")

### GroupedBarChart 컴포넌트
- SVG `<Rect>`로 그룹 막대 차트 구현
- 월별 수입(초록)/저축(파랑)/지출(빨강) 3개 바 그룹
- Y축 자동 스케일링 (최대값 기준 4단계 눈금)
- X축 월 라벨 (1월~12월)
- 수평 스크롤 지원
- 범례 (수입/저축/지출)
- 빈 상태 처리

### StatisticsScreen 통합
- 월별 뷰: DonutChart (카테고리별 지출 비율) 표시
- 연도별 뷰: GroupedBarChart (월별 수입/저축/지출 비교) 표시
- `useEffect` 추가로 기간 변경 시 데이터 재로드
- `savingsService` 연동 (월별 저축 데이터)
