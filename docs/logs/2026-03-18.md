# 2026-03-18 시트 분리 리팩토링

## 구현 내용

`(필수)설정 시트` 하나에 모든 설정 데이터가 혼재되어 batchUpdate 400 에러가 반복 발생하던 문제를 해결하기 위해 시트를 분리.

### 새 시트 구조
| 시트명 | 데이터 | 범위 |
|--------|--------|------|
| `설정-카테고리` | 지출 대분류+소분류 매트릭스 | `A1:P20` (20행×16열) |
| `설정-결제수단` | 신용카드(A열), 체크/현금(B열) | `A1:A10`, `B1:B10` |

### 변경 사항
- **상수 추가**: `CATEGORY_MATRIX`, `PAYMENT_MAX_ROWS`, `SHEET_NAMES.SETTINGS_CATEGORIES/PAYMENTS`
- **범위 변경**: `CELL_RANGES.CATEGORIES/PAYMENT_CREDIT/PAYMENT_DEBIT` → 새 시트 기반 A1 범위
- **exportSettings**: 패딩 10×10 → 16×20 (상수 사용), 결제수단 5→10행
- **importAll**: 연도 셀 읽기 제거, `year?` 파라미터 추가, 인덱스 조정 (0→카테고리, 1→신용, 2→체크)
- **template.xlsx**: `설정-카테고리`, `설정-결제수단` 시트 탭 추가
