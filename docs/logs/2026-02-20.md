# 2026-02-20 작업 로그

## 구현 내용: 엑셀 템플릿 전체 반영 (Phase 1~8)

엑셀 템플릿(`docs/template/template.xlsx`)의 모든 기능을 앱에 반영하는 8단계 구현 완료.

### Phase 1: 기반 — 타입 확장 + ServiceRegistry + SubCategoryService
- `src/types/index.ts` 확장: SubCategory, SavingsProduct, BankAccount, IncomeTarget, MonthlyCategoryMatrix, EnhancedMonthlySummary 등 15+ 타입 추가
- `ServiceRegistry.ts` 도입: HomeScreen 싱글톤 패턴 → 중앙 서비스 레지스트리 리팩토링
- `SubCategoryService` 신규: 소분류 CRUD (Map 기반)

### Phase 2: 카테고리 계층 — 대분류/소분류 시스템
- `defaultCategories.ts`: 10개 지출 대분류 + 소분류, 5개 수입 카테고리, 3개 결제수단
- `SeedService`: 기본 데이터 자동 세팅 (중복 방지)
- `CategoryGroupItem`: 아코디언 형태 대분류/소분류 컴포넌트
- `AddTransactionScreen`: 2단계 카테고리 선택 (대분류 → 소분류)
- `SettingsScreen`: 카테고리 그룹 관리, 소분류 추가/삭제
- `TransactionItem`: "대분류 > 소분류" 표시

### Phase 3: 결제수단 유형 분류
- `PaymentMethodService.getByType()` 추가
- `PaymentMethodType`: credit/debit/cash/account 분류

### Phase 4: 수입 목표 관리
- `IncomeTargetService`: 월별 수입 목표 CRUD
- `IncomeSummaryCard`: 수입 목표 vs 실적 비교 테이블

### Phase 5: 저축 관리
- `SavingsService`: 저축 상품 CRUD + 상태/금액 조회
- `SavingsProductItem`, `SavingsSummaryCard` 컴포넌트

### Phase 6: 은행 계좌/자산 관리
- `BankAccountService`: 계좌 CRUD + 등급별/총자산 조회
- `AssetOverview`: 금융기관 등급별 자산 현황 컴포넌트

### Phase 7: 연간 대시보드
- `TransactionService.getAnnualCategoryMatrix()`: 12개월 × 카테고리 지출 매트릭스
- `AnnualDashboard`: 가로 스크롤 연간 매트릭스 테이블

### Phase 8: 월간 상세 요약 강화
- `TransactionService.getEnhancedMonthlySummary()`: 저축률, 잔여현금, 급여대비 저축률
- `SummaryCard` 확장: 저축/저축률/잔여현금 optional 표시
