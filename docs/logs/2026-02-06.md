## TODO: 거래 등록 시 날짜 선택 기능

### 요구사항
- 거래(수입/지출) 등록 시 날짜를 선택할 수 있어야 함
- 현재는 자동으로 오늘 날짜가 설정됨

### 구현 방식
**빠른 선택 버튼 + 달력 픽커 모두 지원**

```
┌────────────────────────────────────┐
│  날짜                              │
├────────────────────────────────────┤
│  [오늘] [어제] [직접 선택]          │
│                                    │
│  선택된 날짜: 2026년 2월 5일 (수)   │
└────────────────────────────────────┘

[직접 선택 클릭 시]
┌────────────────────────────────────┐
│  ◀  2026년 2월  ▶                  │
├────────────────────────────────────┤
│ 일  월  화  수  목  금  토          │
│                         1          │
│  2   3   4  [5]  6   7   8         │
│  ...                               │
└────────────────────────────────────┘
```
